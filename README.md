# JLPT N4 Past Paper Review

Static review site for JLPT N4 past papers.

## Use

Open `index.html` in a browser, choose an exam year/session, then answer questions one by one.

## Included

- 2025 July N4
- 2025 December N4
- 2024 July N4
- 2023 July N4
- 2023 December N4

Progress is stored locally in the browser.
